# データ型とリテラル

## 代表的なデータ型

| データ種別 | 区分   | 代表的なデータ型名            |
| ---------- | ------ | ----------------------------- |
| 数値       | 整数値 | INTEGER 型                    |
| 数値       | 小数   | DECIMAL 型、REAL 型           |
| 文字列     | 固定長 | CHAR 型                       |
| 文字列     | 可変長 | VARCHAR 型                    |
| 日付と時刻 |        | DATETIME 型、DATE 型、TIME 型 |
